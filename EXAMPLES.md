# F-Map Examples

## Example 1

Goal:
Build an AI Trading App

Systems:

- Frontend
- Backend
- Database
- Authentication
- Hosting

Dependencies:

Frontend -> Backend
Backend -> Database
Authentication -> Database

Risks:

- API costs
- Security
- Scaling
