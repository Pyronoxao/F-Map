# F-Map Core Agents

## Scout

Purpose:
Understand the user's idea.

Outputs:
- Requirements
- Features
- User types
- Complexity estimate

---

## Architect

Purpose:
Design the system.

Outputs:
- Frontend
- Backend
- Database
- Infrastructure

---

## Auditor

Purpose:
Estimate costs.

Outputs:
- Development cost
- Monthly cost
- API cost
- Scaling cost

---

## Optimizer

Purpose:
Find cheaper and more efficient solutions.

Outputs:
- Cost reductions
- Simpler architectures
- Alternative providers

---

## Builder

Purpose:
Generate deployment plans.

Outputs:
- Folder structure
- Database schema
- Final builder prompt

---

## Foreman

Purpose:
Monitor live projects.

Outputs:
- Performance reports
- Error reports
- Cost reports

---

## Investor

Purpose:
Evaluate commercial viability.

Outputs:
- Market size
- Competitor analysis
- Revenue opportunities
