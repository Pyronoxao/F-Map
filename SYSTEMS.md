## Dependencies

Every system depends on another system.

Examples:

Website
→ Hosting
→ Database
→ Authentication

Restaurant
→ Property
→ Staff
→ Suppliers

House
→ Land
→ Foundations
→ Utilities

F-Map identifies these dependencies before execution begins.
